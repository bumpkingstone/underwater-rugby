# 水下橄榄球 (Underwater Rugby) - 设计规格

## 1. 项目概述

- **项目名称**：Underwater Rugby
- **类型**：3D等距视角体育竞技游戏
- **风格**：赛博朋克（霓虹发光、深色背景、六边形容器）
- **玩法**：玩家控制一名队员，其他队员由AI控制，先进5球获胜
- **技术栈**：Three.js（单HTML文件架构）

---

## 2. 视觉设计

### 场景
- **场地**：六边形透明容器，边界有发光线条（赛博朋克风格）
- **背景**：深色海底 + 动态气泡粒子 + 光线柱效果
- **光照**：霓虹色点光源，营造科技感氛围

### 角色（球员）
- 3D球形/胶囊形球员模型
- 发光轮廓（外发光效果）
- 队伍颜色：
  - 队伍A：青色 (Cyan) 霓虹
  - 队伍B：品红 (Magenta) 霓虹
- 玩家控制的球员有额外高亮提示

### 球
- 发光球体
- 霓虹色
- 移动时有拖尾粒子效果

### UI元素
- 比分显示（顶部）
- 当前控制球员指示
- 能量条（冲刺用）
- Tab切换界面（显示队友位置）

---

## 3. 游戏机制

### 操控
- **WASD / 方向键**：3D空间移动（上下左右前后）
- **鼠标/摇杆**（可选）：调整视角方向
- **Tab（按住）**：显示所有队友位置 → 按1-3选择控制
- **空格键**：冲刺（消耗能量，能量随时间回复）

### AI系统（可升级模块）
- 简单状态机实现
- AI行为：追球 → 进攻/防守 → 射门
- 每个AI独立运行
- **可升级点**：状态机 → 目标导向AI → 规则驱动AI

### 比赛规则
- 先进5球获胜
- 球门位于场地两端
- 边界碰撞反弹
- 进球后重置位置

---

## 4. 音效系统

### 背景音乐
- 电子风格循环BGM
- 低频节拍营造水下氛围
- 可内嵌或使用CDN链接

### 进球庆祝音乐
- 进球时播放庆祝音效
- 队伍专属庆祝音效

### 其他音效
- 球员碰撞声
- 球击中边界声
- 冲刺音效

---

## 5. 技术架构（单文件HTML）

```
├── Game Loop (60fps)
│   ├── Input Handler (键盘事件 + 鼠标)
│   ├── Physics Engine
│   │   ├── 碰撞检测（球员-球、球员-边界、球员-球员）
│   │   ├── 3D边界反弹
│   │   └── 水下阻力模拟
│   ├── Camera Controller
│   │   ├── 等距视角（可旋转）
│   │   └── 跟随玩家控制的球员
│   ├── AI Controller
│   │   └── 状态机（追球/进攻/防守/射门）
│   ├── Renderer
│   │   ├── Three.js Scene
│   │   ├── 相机（等距视角，可旋转）
│   │   ├── 灯光（霓虹光源）
│   │   └── 粒子效果（气泡、光线）
│   └── UI Manager
│       ├── 比分
│       ├── 能量条
│       └── Tab切换界面
```

---

## 6. 可升级模块清单

以下模块可在基础版完成后升级：

1. **AI系统**
   - 当前：简单状态机
   - 升级：目标导向AI → 规则驱动AI

2. **物理引擎**
   - 当前：基础碰撞
   - 升级：流体力学模拟（水下阻力）

3. **视觉效果**
   - 当前：基础霓虹发光
   - 升级：全局光照、后期处理、更多粒子效果

4. **游戏模式**
   - 当前：vs电脑
   - 升级：双人同屏对战 → 在线多人对战

5. **音效系统**
   - 当前：基础音效
   - 升级：3D空间音频、语音播报

## 7. 开发阶段

### Phase 1: 基础框架
- Three.js 场景搭建
- 等距视角相机
- 球员和球体渲染

### Phase 2: 核心玩法
- 玩家3D空间移动控制
- 3D物理碰撞（球员-球、球员-边界、球员-球员）
- 水下阻力模拟
- 进球判定

### Phase 3: AI系统
- AI状态机
- AI追球和射门

### Phase 4: 完善
- UI（比分、能量条）
- 音效
- Tab切换功能
- 六边形场地
